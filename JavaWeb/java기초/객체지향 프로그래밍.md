# 객체지향 프로그래밍

### 객체(Object)란?

> - 의사나 행위가 미치는 대상 (사전적 의미)
> - 구체적, 추상적 데이터의 단위 

<br>

### 객체 지향 프로그래밍 (Object Oriented Programming)

> - 객체를 기반으로 하는 프로그래밍
> - 객체를 정의하고, 객체의 기능을 구현하며 객체간의 협력을 구현하는 방식
> - 시간의 흐름에 따르는 절차 지향 프로그래밍과는 다르게 객체간의 연관관계를 생각



### 클래스 (Class)

> - 객체의 청사진이라 할 수 있다.
> - 멤버 변수와 메서드를 가질 수 있음.
